Patterns & Sequences

Arithmetic Sequences
Ordinary 6 questions · 21 marks total
✖️
📘 What is this topic?

ARITHMETIC SEQUENCE: constant difference d between terms. nth TERM: Tₙ=a+(n−1)d. SUM of n terms: Sₙ=n/2×[2a+(n−1)d].

💡 Analogy — How to picture it

A staircase with equal step heights. First step at height a, each step rises by d. The 10th step is at height a+9d. The formula is 'start + steps taken × step height'.

🔍 Worked Example
For 4,7,10,13,… find T₁₀ and S₁₀
  1. a=4, d=3
  2. Tₙ=4+(n−1)×3=3n+1
  3. T₁₀=31
  4. S₁₀=10/2×[8+27]=5×35=175
⚠️ Common Mistakes
  • Formula has (n−1)×d, NOT n×d
  • To check if a value is IN the sequence: set Tₙ=value, solve for n. Whole number=yes.
  • Sum formula uses a and d, not the last term
📌 Key Formulae
Tₙ=a+(n−1)dSₙ=n/2×[2a+(n−1)d]d=T₂−T₁
🎯 SEC Exam Insight: Sequence questions: common SEC pattern is (a) find Tₙ; (b) find a specific term; (c) 'is x a term?' Set Tₙ=value, solve for n. If n is a whole number, yes — it is in the sequence.
Q1 Find the next three terms: 6, 11, 16, 21, … 2 marks
💡 Find the common difference (d=11−6=5) and add it each time.
✅ Answer: d=5; next: 26, 31, 36
Q2 For 4,7,10,13,… find Tₙ and T₁₀. 3 marks
💡 a=4, d=3. Use Tₙ=a+(n−1)d.
✅ Answer: Tₙ=3n+1; T₁₀=31
Q3 Is 2 a term in the sequence 100, 93, 86, 79, …? 3 marks
💡 Find Tₙ, set it equal to 2, solve for n. If n is a whole number, yes!
✅ Answer: Tₙ=107−7n. Set 107−7n=2 → n=15. Yes, 2 is the 15th term.
Q4 Find the sum of the first 15 terms of: 3, 8, 13, 18, … 4 marks
💡 Use the sum formula Sₙ=n/2×[2a+(n−1)d].
✅ Answer: a=3, d=5. S₁₅=15/2×[6+70]=7.5×76=570
Q5 Matchstick pattern: Diagram 1=4, D2=7, D3=10 sticks. How many for Diagram 20? 4 marks
💡 Find a and d from the pattern, then find T₂₀.
✅ Answer: a=4, d=3; Tₙ=3n+1; T₂₀=61 sticks
Q6 3rd term=11 and 7th term=27. Find first term, d, and Tₙ. 5 marks
💡 Write two equations for T₃ and T₇, then solve simultaneously.
✅ Answer: T₃:a+2d=11; T₇:a+6d=27. Subtract:4d=16→d=4;a=3. Tₙ=4n−1
