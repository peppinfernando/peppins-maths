Probability

Counting Principles & Expected Value
Higher 6 questions · 21 marks total
📊
📘 What is this topic?

FUNDAMENTAL COUNTING PRINCIPLE: m ways × n ways = m×n ways. PERMUTATIONS (order matters): nPr=n!/(n−r)! COMBINATIONS (order doesn't matter): nCr=n!/(r!(n−r)!).

EXPECTED VALUE: E(X)=Σ[x×P(x)]. Fair game: E(X)=0.

💡 Analogy

Permutations=PASSWORDS (ABC≠CAB). Combinations=COMMITTEES (same people, different order = same committee). Use ⁿCᵣ on your calculator!

🔍 Worked Example
8 people: (a) elect president+sec+treasurer (b) form committee of 3
  1. (a) ORDER MATTERS: ⁸P₃=8×7×6=336
  2. (b) ORDER DOESN'T MATTER: ⁸C₃=336/6=56
⚠️ Common Mistakes
  • Always ask: does ORDER matter? Yes→Permutation. No→Combination
  • 0!=1 (by definition)
  • Negative E(X) means expected LOSS
📌 Key Formulae
n!=n×(n−1)×…×1nPr=n!/(n−r)! [order matters]nCr=n!/(r!(n−r)!) [order doesn't]E(X)=Σ[x×P(x)]
🎯 SEC Exam Insight: Counting principles: ask 'does order matter?' Roles (president, secretary) → order matters → Permutation nPr. Committees/groups → order doesn't matter → Combination nCr.
