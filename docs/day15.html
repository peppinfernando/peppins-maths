Day 15 of 45 · Algebra

Functions

Notation, Evaluation & Graphs
Ordinary→Higher 6 questions · 20 marks total
✖️
📘 What is this topic?

A FUNCTION maps each input (x) to exactly ONE output f(x). Evaluate by replacing every x with the given number and using BEMDAS.

GRAPH: make a table of values, plot points, draw smooth curve (quadratic) or straight line (linear). Roots=where f(x)=0. Vertex (min/max) of parabola: x=−b/2a.

💡 Analogy — How to picture it

A function is like a vending machine — press button x, get exactly one item f(x). It ALWAYS gives the same item for the same button.

🔍 Worked Example
Draw f(x)=x²−3x−4 for −1≤x≤4
  1. Table: x=−1:0, x=0:−4, x=1:−6, x=2:−6, x=3:−4, x=4:0
  2. Plot all points on a grid
  3. Draw smooth parabola — NEVER a V-shape!
  4. Roots at x=−1 and x=4
⚠️ Common Mistakes
  • Make a TABLE of at least 5 values — never sketch freehand
  • ROOTS=where f(x)=0, MINIMUM=lowest point — different things!
  • f(−3)=(−3)²=9, NOT −3²=−9
📌 Key Formulae
Evaluate: replace x with given valueRoots: solve f(x)=0Vertex: x=−b/2a
🎯 SEC Exam Insight: Functions: always make a TABLE of at least 5 values before drawing — never sketch freehand. Roots and turning point are common part (c) questions worth significant marks.
Q1 If f(x)=2x²−5, find f(3). 2 marks
💡 Replace x with 3 and calculate.
✅ Answer: 2(9)−5=13
Q2 If g(x)=3x+1, find g(−2). 2 marks
💡 Put −2 in brackets when substituting.
✅ Answer: 3(−2)+1=−5
Q3 Find where f(x)=g(x) for f and g from Q1 and Q2 (to 2 d.p.). 3 marks
💡 Set the two functions equal and solve the resulting quadratic.
✅ Answer: 2x²−5=3x+1 → 2x²−3x−6=0 → x≈2.64 or −1.14
Q4 Draw h(x)=x²−3x+1 for −2≤x≤3 using a table of values. 4 marks
💡 Calculate y for each x value in the range, then plot.
✅ Answer: x:−2(11),−1(5),0(1),1(−1),2(−1),3(1). Plot and draw smooth parabola.
Q5 Using your graph from Q4, find the minimum value of h(x). 4 marks
💡 The minimum is the lowest point of the parabola (turning point).
✅ Answer: Min at x=1.5; h(1.5)=2.25−4.5+1=−1.25
Q6 From your Q4 graph, estimate the roots of h(x)=0 to 1 d.p. 5 marks
💡 Roots are where the graph crosses the x-axis (y=0).
✅ Answer: Roots ≈ x≈0.4 and x≈2.6 (where graph crosses x-axis)
