Day 31 of 45 · Statistics

Statistics

Charts & Diagrams
Foundation→Ordinary 6 questions · 21 marks total
📊
📘 What is this topic?

BAR CHART: gaps between bars, discrete/categorical. HISTOGRAM: no gaps, continuous, y-axis=frequency density=freq÷class width.

PIE CHART: sector angle=(freq÷total)×360°. STEM-AND-LEAF: shows actual data in order; stem=tens digit, leaf=units.

💡 Analogy — How to picture it

A histogram is like a city skyline — each building (bar) covers a range. A pie chart divides 360° like sharing a round pizza proportionally.

🔍 Worked Example
Data 14,22,19,31,27,15,23,18,28,11. Stem-and-leaf + median.
  1. 1|1 4 5 8 9
  2. 2|2 3 7 8
  3. 3|1
  4. Ordered: 11,14,15,18,19,22,23,27,28,31
  5. Median=(19+22)/2=20.5
⚠️ Common Mistakes
  • Histogram y-axis: equal widths→frequency ok; different widths→MUST use frequency density
  • Pie chart: all angles must SUM to 360°
  • Bar chart=gaps (discrete); histogram=no gaps (continuous)
📌 Key Formulae
Pie sector=(f/total)×360°Freq density=freq÷class width
🎯 SEC Exam Insight: Histograms: if class widths are EQUAL, y-axis can show frequency. If widths DIFFER, y-axis MUST show frequency density=freq÷class width. Check widths before drawing.
Q1 Class votes: A(40),B(30),C(20),D(30). Total=120. Find sector angle for each. 2 marks
💡 Angle=(frequency÷total)×360°.
✅ Answer: A:120°, B:90°, C:60°, D:90°. Sum=360° ✓
Q2 Draw a stem-and-leaf for: 23,18,35,29,20,15,24,31,27,12. 3 marks
💡 Stem=tens digit, leaf=units digit. Order the leaves on each stem.
✅ Answer: 1|2 5 8; 2|0 3 4 7 9; 3|1 5
Q3 Using your stem-and-leaf from Q2, find the median. 3 marks
💡 The stem-and-leaf already puts data in order — find the middle value.
✅ Answer: Ordered: 12,15,18,20,23,24,27,29,31,35 (n=10). Median=(23+24)/2=23.5
Q4 Ages: 0−10(6),10−20(14),20−30(20),30−40(8),40−50(2). Find frequency density for each. 4 marks
💡 Frequency density=frequency÷class width. All widths are 10 here.
✅ Answer: All class widths=10. FD: 0.6, 1.4, 2.0, 0.8, 0.2
Q5 From a histogram, how do you find the number of people in a specific age range? 4 marks
💡 In a histogram, AREA of bar=frequency.
✅ Answer: Frequency=frequency density×class width. Multiply bar height by class width.
Q6 Using Q4 data (n=50): find modal class and estimate median age. 5 marks
💡 Modal class has highest frequency. Find where the 25th value falls in cumulative frequency.
✅ Answer: Modal class: 20−30 (freq=20). Median=25th value. CumFreq: 6,20,40. Median in 20−30 class. Est≈22.5
